# lucide-react 全站图标化（构思，2026-04-30）

**范围**：本仓库 `app/` 内已有「+ / ✕ / 文字删除」等占位与操作控件；**不替换** `ContextUsageBar` 的环形用量 SVG（专用可视化）。

## 已明确采纳

| 想法                                                          | 理由                                        |
| ------------------------------------------------------------- | ------------------------------------------- |
| 側栏与聊天 composer 的操作改为图标 + `aria-label`             | 对齐常见 AI 产品信息密度，不丢可访问性      |
| `next.config` 开启 `optimizePackageImports: ['lucide-react']` | 减少 barrel import 包体（Vercel/Next 惯例） |
| 设置页「添加模型」类按钮：`Plus` + 文案                       | 比纯「+」字符更一致                         |
| 模型列表删除：`Trash2` + 现有 `aria-label`                    | 与侧栏删除视觉语言统一                      |

## 刻意不做 / defer

| 想法                            | 理由                                   |
| ------------------------------- | -------------------------------------- |
| 把所有 section 标题都换成大图标 | 易喧宾夺主；仅设置页 h2 小图标点缀即可 |
| 用 lucide 重画上下文圆环        | 无收益，增加维护                       |
